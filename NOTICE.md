# Notice and acknowledgements

The project concept was informed by public experiments around Apple network-location behavior,
including `Yu9191/wloc`. That repository did not expose a recognized open-source license when this
project was prepared, so no source code from it is included here.

The protobuf framing, guarded no-fix behavior, geometry-preserving translation, scoped TLS
interceptor, installer, and tests in this repository are maintained as an independent
implementation.

Home-Location-Endpoint downloads but does not redistribute Xray-core source or binaries in this
repository. Xray-core remains subject to its own license. Realm is mentioned only as an optional
external TCP relay. City-boundary results are derived from OpenStreetMap contributors through the
Nominatim service and must retain OpenStreetMap attribution.
